🔧 How to Use Angular DevTools:
  1. Install Angular DevTools:
    • Chrome: Search "Angular DevTools" in Chrome Web Store
    • Firefox: Search "Angular DevTools" in Firefox Add-ons
  2. Open Developer Tools (F12) → Find "⭕ Angular" tab
  3. Explore the component tree in the left panel
  4. Click on components to see their properties and state
  5. Interact with the buttons above and watch state changes in real-time
🔍 What to Inspect:
  • Component Tree: Parent-child relationship hierarchy
  • Properties: @Input(), @Output(), and local component state
  • Injector: Services and dependencies
  • Router: Current route and route parameters
💡 Debug Techniques:
  • Watch how @Input() properties change when parent updates data
  • See @Output() events trigger when child components emit
  • Inspect component state before and after user interactions
  • Use the "Profiler" tab to analyze performance and change detection
